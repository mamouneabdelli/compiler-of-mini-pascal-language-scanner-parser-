## **1. Backus Normal Form (BNF)**

### **BNF Table**

| Representation   | Explanation                                                                                                                  |
| ---------------- | ---------------------------------------------------------------------------------------------------------------------------- |
| `::=`            | Sometimes `=` is used                                                                                                        |
| `<Non-terminal>` | Non-terminals are written between angle brackets. The paper uses both `<` and `<=` but standard notation is `<Non-terminal>` |
| **Terminals**    | Written in lowercase                                                                                                         |
| `ab`             | Concatenation: represents the sequence `a` followed by `b`                                                                   |
| `a / b`          | Choice: either `a` or `b`                                                                                                    |
| `[a]`            | Option: zero or one occurrence of `a`                                                                                        |
| `a*` or `{a}`    | Repetition: any number of `a`, including none                                                                                |

## 3. Grammar Definition of the MiniPascal-Fr Language

```
<ProgrammePascal> ::= programme <NomProgramme> ; <Corps> .

<Corps> ::= [<PartieD√©finitionConstante>] [<PartieD√©finitionVariable>] <InstrComp>

<PartieD√©finitionConstante> ::= constante <D√©finitionConstante> {<D√©finitionConstante>}
<D√©finitionConstante> ::= <NomConstante> = <Constante> ;

<PartieD√©finitionVariable> ::= variable <D√©finitionVariable> {<D√©finitionVariable>}
<D√©finitionVariable> ::= <GroupeVariable> ;
<GroupeVariable> ::= <NomVariable> {, <NomVariable>} : <NomType>

<NomVariable> ::= <Nom>
<Nom> ::= <lettre> <chiffre> {<lettre> | <chiffre>}
<Lettre> ::= "A" | "B" | ... | "Z" | "a" | ... | "z"
<Chiffre> ::= 0 | 1 | ... | 9

<NomType> ::= entier | reel

<NomConstante> ::= <Nom>
<Constante> ::= <Nombre> | <NomConstante>
<NomProgramme> ::= <Nom>

<Nombre> ::= <Chiffre> {<Chiffre>} | <Chiffre> <Chiffre> {<Chiffre>} . <Chiffre> {<Chiffre>}

<InstrComp> ::= debut <Instruction> {; <Instruction>} fin

<Instruction> ::= <InstructionAffectation>
                | <InstructionSi>
                | <InstructionTantque>
                | <InstructionRepeter>
                | <InstructionPour>
                | <Vide>

<InstructionAffectation> ::= <NomVariable> := <Expression>

<Expression> ::= <ExpressionSimple> [<OperateurRelationnel> <ExpressionSimple>]

<OperateurRelationnel> ::= = | <> | < | > | <= | >=

<ExpressionSimple> ::= [<OperateurSigne>] <Terme> {<OperateurAddition> <Terme>}

<OperateurSigne> ::= + | -
<OperateurAddition> ::= + | - | ou

<Terme> ::= <Facteur> {<OperateurMult> <Facteur>}

<OperateurMult> ::= * | div | mod | et

<Facteur> ::= <Constante> | <NomVariable> | ( <Expression> )

<InstructionSi> ::= si <Expression> alors <Instruction> [sinon <Instruction>]

<InstructionTantque> ::= tantque <Expression> faire <Instruction>

<InstructionRepeter> ::= repeter <Instruction> jusqu‚Äôa <Expression>

<InstructionPour> ::= pour <NomVariable> allant de <Constante> a <Constante> [pas <Constante>] faire <Instruction>

<Vide> ::= ''
```
